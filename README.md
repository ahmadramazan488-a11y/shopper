# Shopper

A simple test ecommerce storefront.

## Features

- Product catalog with demo items
- Category filters
- Add to cart
- Quantity controls and remove button
- Cart subtotal and free shipping
- Demo checkout/order form
- Static HTML, CSS, and JavaScript for easy Cloudflare deployment

## Deployment test

This tiny README change is only here to confirm Cloudflare creates a PR preview and deploys again after merge.

This is a test project only. It does not collect real payments or send real orders.
